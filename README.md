# Тестовое задание

## Установка и запуск

- Использованы localstorage + IndexDb
- Вычисления и аггрегация вынесены на сторону веб-воркера
- Подгрузка данных идет с помощь SSRM
- Данные сохраняются и действительные 24 часа
- Первая загрузка данных продолжительная -> происходит агрегация данных и сохранение, последующие загрузки берутся из кеша( IndexDb ) и быстро отображаются
- Интерфейс не блокируется

- Что имелось ввиду непонятно...(смещение массива метрик??)

### 2. Корректная обработка периодов

Периоды статистики в листах отличаются, заканчиваются на `lastUpdate` и идут по убыванию. Например, если сегодня `lastUpdate` 01.01.2026, то это первый элемент, для второго дата будет 31.12.2025 и т.д. Нужно корректно свести данные.
